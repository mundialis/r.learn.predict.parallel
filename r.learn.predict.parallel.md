## NAME

***r.learn.predict.parallel*** - Toolset for parallel running
r.learn.predict.

## KEYWORDS

[r.learn.predict](r.learn.predict.md)

## DESCRIPTION

The *r.learn.predict.parallel* toolset consists of the following
modules:

[r.learn.parallel.predict](r.learn.parallel.predict/r.learn.parallel.predict.md)  
compute [r.learn.predict](r.learn.predict.md) in parallel with
[r.learn.predict.worker](r.learn.predict.worker/r.learn.predict.worker.md)

## REQUIREMENTS

[r.learn.predict](r.learn.predict.md) and Python3-scikit-learn
